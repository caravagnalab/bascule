# COSMIC catalogue for SBS filtered (version 3.4, GRCh37)

Filtered version of the COSMIC catalogue. Only validated signatures are
present. For each signature (besides SBS5) the contexts with probability
mass below 0.02 are set to 0 and the probabilities are re-normalized
over contexts.

## Usage

``` r
data(COSMIC_sbs_filt)
```
