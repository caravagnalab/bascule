# Degasperi DBS catalogue

DBS signatures extracted and released in "Substitution mutational
signatures in whole-genome–sequenced cancers in the UK population."
Degasperi et al.Science 376, Issue 6591, 2022. PMID: 35949260

## Usage

``` r
data(Degasperi_DBS)
```

## Format

An object of class `data.frame` with 39 rows and 78 columns.
