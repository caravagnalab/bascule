# Degasperi SBS catalogue

SBS signatures extracted and released in "Substitution mutational
signatures in whole-genome–sequenced cancers in the UK population."
Degasperi et al.Science 376, Issue 6591, 2022. PMID: 35949260

## Usage

``` r
data(Degasperi_SBS)
```

## Format

An object of class `data.frame` with 120 rows and 96 columns.
